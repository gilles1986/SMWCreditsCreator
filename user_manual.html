Overview

The SMW Credits Creator converts a list of contributor names into Map16 tiles that can be imported directly into Lunar Magic. The application has two main tabs:

  • Credits — Set up your text source, appearance options, Map16 settings, and generate the output file.
  • Mapping — Define or edit which graphic tile IDs correspond to each text character, load ExGFX graphics, and pick tiles visually.

Credits Tab

This is where you configure what to generate and how it should look.

1. Credits Content

Choose how your contributor names are provided:

  • File — Select a file via Browse…. Supported formats:
    • .txt — One author per line.
    • .json — Exported from Saphros SMW Credits Manager.
  • Direct Text — Paste names directly into the text box (one per line).
Switch between File and Direct Text mode using the toggle button at the top of the section.

2. Appearance & Font

  • Font Mapping — Shows which mapping file is loaded. Use Load Mapping (.json) to load a custom one, or ↺ to reset to defaults.
  • Capitalize Text — Converts all text to UPPERCASE automatically.
  • Optimize Columns — Fits two short names side-by-side into one row.
  • Add Empty Line After Section — Inserts a blank row between credit sections.
  • Font Size — Choose 8x8, 8x16 (tall), or 16x16 (full block).

3. Map16 Settings

  • Blank Tile ID — The tile used for empty space (default: 0F8).
  • Act As — The tile behavior, e.g. 0025 = Air, 0130 = Cement, 002B = Coin.
  • Start Page (Hex) — The Map16 page where tiles begin (e.g. 60).
  • Priority (On Top) — Renders tiles on top of sprites.
  • Section Order — Controls the order of credit sections:
    • predefined — Fixed order (Sprites, UberASM, …).
    • alphabetical — Sorted A–Z.
    • none — File order as-is.

Output

Click Save .map16 to generate the Map16 file. Progress and status messages appear in the Output Log at the bottom.

Mapping Tab

Defines which ExGFX tile ID corresponds to which text character.

Character Map

  • The left column lists every character and its current tile ID(s).
  • Click the 🎯 target icon next to a character, then click a tile in the graphics preview to assign it.
  • Use ⇄ BG3 to convert all IDs between BG3/non-BG3 format.
  • Press ℹ for a quick-reference of the mapping format.

Graphics Preview

  • Load Graphics (.bin) — Load your ExGFX binary file to see and pick tiles.
  • Load Palette (.bin) — Load a palette file for correct colors. You can also select a palette row (0–7).
Auto-Load: Place a file named palette.pal in the tool folder to load colors automatically on start.

Configuration

The Mapping tab also shows the shared configuration values (Act As, Start Page, Tile Size, Palette, Priority) so you can adjust them from either tab — changes are synced.

Export & Import into Lunar Magic

Generating the File

  1. Go to the Credits tab.
  2. Set your source (file or direct text) and adjust settings.
  3. Click Save .map16 and choose a filename.

Importing into Lunar Magic

Important: The generated file contains the target page information, so tiles are placed on the correct page automatically.
  1. Open Lunar Magic.
  2. Open the 16×16 Tile Map Editor.
  3. Navigate to the page you selected (e.g. Page 0x60).
  4. Press F3 (or File → Import Map16).
  5. Select your generated .map16 file.

The tiles should appear instantly on the selected page.
